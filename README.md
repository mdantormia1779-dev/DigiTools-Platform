Project Name: DigiTools-Platform buying website

Description:
DigiTools is a modern and user-friendly web application where users can explore various digital tools and easily add them to a cart . it provides a smooth shoping experience with dynamic cart functionality and responsive design

Technology Used:
1.React.js
2.Tailwind css
3.Daisy UI
4.JavaScript(Es6)
5.React-Toastify
6.JSON

Features:
1.add to card system with live cart count update in Navbar
2.Toggle system between products and card section
3.remove items and clear cart using proceed to checkout button
